```
아래 회의 메모를 요약해줘.

요약 형식:
1. 회의 개요 (일시, 참석자, 주제)
2. 주요 논의 내용 (3~5줄)
3. 결정된 사항 (bullet point)
4. 액션 아이템 (담당자 + 기한 포함)
5. 다음 회의 일정

---
[회의 메모 원문]

일시: 2025년 7월 10일 (목) 오후 2시
참석자: 김민준(PM), 이서연(개발), 박지호(디자인), 최유나(마케팅)

- 김민준: 오늘은 8월 출시 예정인 앱 리뉴얼 관련해서 진행 상황 점검하려고 모였어요.
- 이서연: 개발은 현재 70% 완료됐고요, 로그인/회원가입 파트는 다음 주 중으로 마무리될 것 같아요. 
  근데 결제 모듈 연동이 생각보다 오래 걸려서, 7월 말까지는 어려울 것 같고 8월 5일 정도가 현실적일 것 같아요.
- 박지호: 디자인은 메인 화면이랑 마이페이지는 완료됐는데, 결제 화면 시안을 아직 못 넘겼어요. 
  개발 일정이 밀리면 저도 이번 주 금요일까지 드릴게요.
- 최유나: 마케팅은 SNS 사전 예고 콘텐츠 준비 중인데, 출시일이 확정돼야 캠페인 일정을 잡을 수 있어요. 
  늦어도 7월 18일까지는 출시일 확정이 필요해요.
- 김민준: 그럼 출시일은 8월 7일로 잠정 확정하고, 7월 18일에 최종 확인하는 걸로 해요. 
  혹시 QA 일정은요?
- 이서연: 개발 완료 후 QA에 최소 5일은 필요해요. 8월 5일 개발 완료면 QA는 8월 6~7일인데 빠듯하긴 해요.
- 김민준: 일단 8월 5일 개발 완료를 목표로 하고, 안 되면 출시일 8월 14일로 미루는 걸 플랜 B로 두죠.
- 박지호: 저는 결제 화면 시안 이번 주 금요일(7/11)까지 드릴게요.
- 다음 회의는 7월 18일 오후 2시로 잡겠습니다.
```
